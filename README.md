# REST-API-FLASK_1

Hi, 

This is a simple REST Api with CRUD functionality, implemented using Flask, Flask-JWT, Flask-RESTful, Flask-SQLAlchemy and Postgres. 

## Endpoints. 
App has 3 database tables: users, stores and items.
